# Pf1 – Flask Microservice (Lab 6.3.6)

## Doel
Een eenvoudige Flask microservice bouwen en runnen met Docker.

## Structuur
- `sample_app.py` – Flask applicatie
- `templates/` – HTML templates
- `static/` – statische bestanden
- `sample-app.sh` – Docker build & run script
- `screenshots/` – bewijs

## Run
```bash
bash ./sample-app.sh
```

## Test
curl http://localhost:5050

## Resultaat

Docker image sampleapp gebouwd

Container samplerunning draait

Flask app bereikbaar via poort 5050

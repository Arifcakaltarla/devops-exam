## Description
This experiment demonstrates the integration of an external REST API in a Python application.  
The script connects to **JokeAPI**, retrieves JSON data, parses the response, and prints a formatted result to the terminal.

This experiment was created as an **own API experiment (Python)** according to the DevOps evaluation criteria.

---

## Used API
- JokeAPI  
- Endpoint: https://v2.jokeapi.dev/joke/Any

---

## Functionality
- Sends an HTTP GET request to a public REST API
- Checks HTTP status codes
- Parses JSON responses
- Handles different response types (`single` / `twopart`)
- Includes timeout and error handling
- Displays formatted output in the terminal

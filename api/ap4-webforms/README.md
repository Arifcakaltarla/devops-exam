# AP4 – API Experiment 2 (Python) – Webforms (Flask + JokeAPI)

## Doel
Een Flask webapplicatie bouwen met een webformulier.
De gebruiker kiest een categorie en de applicatie haalt een joke op via JokeAPI.

## Bestanden
- `app.py` : Flask applicatie
- `templates/index.html` : HTML webform
- `screenshots/` : bewijs (screenshots)

## Vereisten
- Python 3
- Flask
- Requests

## Starten van de applicatie
```bash
cd ~/devops-exam/api/ap4-webforms
pip3 install flask requests
python3 app.py
http://localhost:5000

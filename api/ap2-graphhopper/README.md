# Ap2 – GraphHopper REST API (Lab 4.9.2)

## Doel
Integreren van een externe REST API (GraphHopper) in een Python applicatie.
Het experiment toont hoe geocoding en routing via een API werken en hoe JSON-responses verwerkt worden.

---

## Overzicht van de onderdelen

### Part 1 – Launch the DEVASC VM
De DEVASC VM werd gestart en gebruikt als vaste labomgeving voor dit experiment.

### Part 2 – Demonstrate the GraphHopper Directions Application
De GraphHopper web applicatie werd bekeken om inzicht te krijgen in:
- routes
- afstand
- reistijd

### Part 3 – Get an API Key
Een GraphHopper API key werd aangemaakt.
De key wordt **veilig gebruikt via een environment variable** en staat niet hardcoded in de code.
